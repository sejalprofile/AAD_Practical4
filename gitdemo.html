Global Config
To set,
git config --global user.name "Your_GitHub_profile_name"
git config --global user.email "Your gmail id"
To get,
git config --global user.name and
git config --global user.email separately or
git config --global --list
To remove,
git config --global --unset user.name
git config --global --unset user.name
